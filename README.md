# English to Bangla Learning Dictionary

## Purpose

English to Bangla Learning Dictionary is a web-based application designed to help users learn and understand English words with their Bangla meanings. The main purpose of this project is to improve vocabulary learning through a simple, interactive, and user-friendly interface. It also demonstrates practical usage of JavaScript and frontend development concepts.
